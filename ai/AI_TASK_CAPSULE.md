## In one paragraph

CodeOutfitters is a real, deployable Next.js 16 marketing site for a US small-business AI automation agency, plus a single-operator password-gated admin tool that takes client intake and generates proposals with the Anthropic API. The app is statically exported to `out/` and targets Cloudflare Pages. All `NEXT_PUBLIC_*` env vars end up in the shipped JS bundle. Forms post to a single shared n8n webhook; bookings additionally write to Supabase. There is no test suite, no CI, no server function, and no middleware.
